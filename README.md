# MindfulBreak

A minimal macOS menu bar app for mindful breaks using a Pomodoro-style timer. Features tech news, programming jokes, a quick game, and tech memes.

## Features
- Menu bar timer and break reminders
- Tech news headlines
- Programming jokes
- Quick Tic-Tac-Toe game
- Tech memes from r/ProgrammerHumor

## Install & Run
- macOS 13.0+ required
- Swift 5.9+ required

```sh
cd /path/to/MindfulBreak
swift run
```

Or open `MindfulBreak.xcodeproj` in Xcode and run.

## Project Structure
- `Sources/` – main app code
- `MindfulBreak/Services/BreakContentProvider.swift` – content loading logic
